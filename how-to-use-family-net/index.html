如何使用家庭环境的浏览器插件

0. 准备条件

  • 在电脑桌面端操作
  • 使用 Chrome 浏览器
  • 关闭自己使用的代理

1. 从 chrome 浏览器进入 WildCard，点击使用代理

2. 安装插件

2.1 安装我们提供的 Chrome 代理插件

安装代理插件

请点击上方按钮打开谷歌应用商店，如下图点击「添加至Chrome」、「添加扩展程序」安装插件。

3. 使用代理

每次使用代理前，请先关闭您自己本地代理。

安装插件后，会自动跳转到插件的配置页面，点击「前往复制」，去 WildCard 复制代理对应的密码。

将在 WildCard 中购买的账号、密码依次填入输入框，点击 「保存」，会自动开启代理。出现如图绿色提示，即可开始使用。

若使用途中切换到其他网页出现如图弹窗，点击取消或重新输入密码即可继续使用。

​

4. 关闭代理

使用时间结束时，会出现如图弹窗，提示输入新的账户密码。

此时在扩展程序中，找到并点击插件图标后，点击关闭按钮即可关闭代理。
